## 安装方法

1. 复制 `https://github.com/yuanmomoya/halo-theme-keepIt`。
2. 进入后台 -> 外观 -> 主题。
3. 点击右下方按钮选择安装主题，随后选择 `远程拉取`。
4. 粘贴复制的链接到远程地址，点击下载即可。
